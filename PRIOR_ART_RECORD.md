## Context of this record

On 2026-07-08 the author applied to Anthropic's open-source / Claude Max program,
presenting SkyLive as the flagship project. This file preserves the submitted
description of the concept as of that date.

## The concept, as disclosed

An **open-source, reproducible** system — buildable by anyone — that live-streams
a skydiver's own point of view to the drop zone as the jump happens
(~14 ms latency, ~4 km range, 5.8 GHz HDZero). Novelty vs. prior state: this
capability existed only in **closed, commercial** broadcast rigs; SkyLive
publishes the **whole chain — helmet sender to open Linux ground station** —
under CC-BY-4.0 so anyone can rebuild it. Deliberate full disclosure (the
patentable ideas were released rather than locked up).
